# GameStore :video_game:

## Projeto de PSI

* Loja de jogos digitais

## Tecnologias Usadas
#### MEAN Stack
* **Frontend**
    * Angular
* **Backend**
    * Node + Express
* **Base de Dados**
    * MongoDB

:rocket:


 ## Comandos para correr os servidores:
 * **Frontend**
    * `ng serve --port 3062 --host 0.0.0.0 --disable-host-check true`
* **Backend**
    * `npm run devstart`
